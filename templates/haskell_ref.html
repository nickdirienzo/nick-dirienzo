So I'm learning Haskell using the online version of Learn You A Haskell For Great Good! and I decided that I need to start taking notes. So, I'll keep this up as a reference sheet for a quick glance of a function or concept.

Haskell Terminal Commands

  • Start the interactive mode in terminal using ghci
  • :set prompt "ghci> " changes the prompt in GHCI from Prelude> to ghci>

Boolean Algebra

  • && means boolean and
  • || means boolean or
  • not negates True and False
  • == means "is equal to"
  • /= means "is not equal to"

General Function Concepts

  • Functions are not followed by parenthesis but rather by just a space and the parameter(s) needed.
  • Function application (calling a function followed by a space) has the highest precedence.
  • Infix function: a function that takes two parameters, but can be rewritten using backticks.
    • div 9 3 can be rewritten to 9 `div` 3 to make it clear which is the dividend and which is the divisor.

  • succ x     returns x+1
  • min x y     returns whichever is less of the two
